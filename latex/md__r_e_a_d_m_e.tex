\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{🧠 Project Overview}{🧠 Project Overview}}\label{md__r_e_a_d_m_e_autotoc_md3}
This mini web application simulates an {\bfseries{Online Book Borrowing / Reservation System}}, built using {\bfseries{Jakarta Servlets}}, {\bfseries{JSP/\+JSTL}}, and {\bfseries{Java\+Beans}}. ~\newline


It follows the {\bfseries{MVC architecture}}\+:
\begin{DoxyItemize}
\item {\bfseries{Model\+:}} Java\+Beans (Book, Borrowed\+List, Library)
\item {\bfseries{View\+:}} JSP pages using JSTL
\item {\bfseries{Controller\+:}} Servlets (Catalog\+Servlet, Borrow\+Servlet, Checkout\+Servlet)
\end{DoxyItemize}

The system allows users to\+:


\begin{DoxyItemize}
\item 📚 View a catalog of available books ~\newline

\item 📖 Reserve (borrow) books ~\newline

\item 🧾 View the list of borrowed books ~\newline

\item 🔁 Return borrowed books ~\newline

\item 💳 Checkout and view total monthly cost ~\newline

\end{DoxyItemize}

\begin{quote}
